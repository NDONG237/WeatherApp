# Weather Website

A simple weather website that displays current weather information for a given city.

## Features

- Search for weather by city name
- Displays temperature, weather description, humidity, and wind speed
- Weather icons for different conditions
- Responsive design

## How to Use

1. Open `index.html` in a web browser
2. Enter a city name in the search box
3. Click "Search" or press Enter
4. View the current weather information

## API

This website uses the api.openweathermap.org API to fetch weather data. An API key is required.

## Files

- `index.html`: Main HTML structure
- `style.css`: CSS styling
- `script.js`: JavaScript for functionality
- `weather/`: Folder containing weather icons (SVG files)
- `message/`: Folder containing message images (PNG files)

## Troubleshooting

- Ensure you have an internet connection to fetch weather data
- If a city is not found, it will display a "City not found" message
- The website works offline for the UI, but needs internet for weather data
